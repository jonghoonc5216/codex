나라장터 사전규격 자동수집 팀공유 사용법

1. 압축파일을 원하는 폴더에 풉니다.

2. config.json을 열고 api_key에 공공데이터포털 인증키를 넣습니다.
   - 공공데이터포털에서 "조달청_나라장터 사전규격정보서비스" 활용신청이 필요합니다.
   - 가능하면 "일반 인증키(Decoding)" 값을 넣으세요.

3. run_g2b_prespec_update.bat을 실행합니다.
   - 마지막 성공 수집일 기준으로 누락분을 수집합니다.
   - 엑셀에는 이전 확인 묶음과 이번 신규 묶음이 함께 표시됩니다.
   - 이번 신규 묶음은 붉은색 굵은 글씨로 표시됩니다.

4. 엑셀 사업명 클릭으로 나라장터 사전규격상세조회까지 이동하려면 최초 1회 브라우저 확장 설치가 필요합니다.
   - install_browser_extension.bat을 실행합니다.
   - 열린 Chrome 확장 프로그램 화면에서 오른쪽 위 개발자 모드를 켭니다.
   - "압축해제된 확장 프로그램을 로드합니다"를 누릅니다.
   - g2b_prespec_deeplink_extension 폴더를 선택합니다.
   - 이미 설치되어 있으면 Chrome 확장 프로그램 화면에서 G2B Prespec Detail Helper를 새로고침합니다.

5. 기본 검색 조건
   - 검색유형: 사전규격공개
   - 업무구분: 일반용역, 기술용역
   - 포함 단어: 공원, 파크, 조경, 관광, 유원지, 체육, 탐방로, 수목원, 정원, 숲, 휴양림, 치유, 야영장, 레포츠, 산림, 체험, 생태, 경관, 도시재생, 바람길, 둘레길
   - 제외 단어: 위탁, 건설폐기물, 임차, 감리, 건설사업관리

참고
- Python과 openpyxl이 필요합니다.
- Codex에서 실행할 때는 압축을 푼 폴더를 기준으로 run_g2b_prespec_update.bat을 실행하면 됩니다.
- 개인 API 키가 들어간 config.json은 공유하지 마세요.
