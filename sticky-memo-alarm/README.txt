스티커 메모 알람 사용법
========================

1. 스티커 메모에 알람 줄을 적습니다.

   예시:
   알람: 18:30
   퇴근 전에 주간보고 보내기

   예시:
   알람: 2026-06-11 18:30
   회의 자료 확인

   예시:
   알람: 내일 오전 9시
   견적서 다시 확인

2. start_sticky_alarm.cmd 를 실행하면 감시가 시작됩니다.

   실행 중에는 스티커 메모의 알람 줄을 1분마다 확인합니다.

3. 시간이 되면 검붉은색 전체화면에 흰색 글씨로 메모 내용이 표시됩니다.
   음성 안내는 없습니다.

4. 알림 닫기:
   확인 버튼, Enter, Space, Esc

5. 종료:
   stop_sticky_alarm.cmd 실행

6. 현재 등록된 알람 확인:
   list_sticky_alarms.cmd 실행

- 새로 작성하거나 수정한 스티커 메모 내용은 기존 내용 아래에 순서대로 추가됩니다.
- 기존에 이미 있던 스티커 메모는 첫 실행 때 기준선으로만 등록하고, Word에 한꺼번에 저장하지 않습니다.
- 작성 중간 내용이 여러 번 저장되지 않도록 같은 내용이 약 1분 동안 유지된 뒤 Word에 추가됩니다.

자동실행
--------

- Windows 시작 시 자동실행되도록 이미 시작프로그램에 등록되어 있습니다.
- 다시 등록해야 하면 install_startup.cmd 를 실행합니다.
- 자동실행을 해제하려면 remove_startup.cmd 를 실행합니다.

주의
----

- 시간이 지난 '알람: 18:30'처럼 날짜가 없는 알람은 다음 날 같은 시간으로 해석됩니다.
- 이미 울린 알람은 같은 메모/같은 시간/같은 내용 기준으로 다시 울리지 않습니다.
- 다시 울리게 하려면 메모 내용이나 알람 시간을 수정하면 됩니다.
