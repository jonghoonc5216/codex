PDF 엑셀변환 사용법

1. 처음 사용하는 컴퓨터
   - 처음실행_필수설치.bat를 한 번 실행합니다.
   - Python 필수 모듈(openpyxl, pdfplumber)을 설치합니다.
   - Python 3이 설치되어 있지 않으면 Python 3 설치 후 다시 실행합니다.

2. 평소 실행
   - PDF_엑셀변환_실행.bat를 실행합니다.
   - 변환 대상 PDF 영역에 PDF 파일을 드래그해서 넣습니다.
   - 대상면적, 기준면적, 승수, 환산계수, 적용비율을 확인하거나 수정합니다.
   - 저장명칭과 저장위치를 확인한 뒤 변환 실행을 누릅니다.

3. 기본값
   - 대상면적: 10,000㎡
   - 기준면적: 10,000㎡
   - 승수: 0.6
   - 환산계수1: 1
   - 환산계수2: 1
   - 적용비율: 100%

4. 결과 엑셀
   - 결과 시트는 금액 시트와 작업량 시트만 생성됩니다.
   - 변환기록 시트와 PDF 원본 추출 시트는 생성하지 않습니다.
   - 같은 이름의 파일이 있으면 덮어쓰지 않고 이름 뒤에 (1), (2)처럼 번호를 붙여 저장합니다.

5. 주의사항
   - PDF는 스캔 이미지가 아니라 표 텍스트를 읽을 수 있는 PDF여야 합니다.
   - 결과 엑셀이 Excel에서 열려 있으면 저장에 실패할 수 있습니다.
   - 저장위치는 로컬 PC 폴더를 권장합니다. 네트워크 저장은 오류가 생길 수 있습니다.
   - template.xlsx, pdf_formula_converter.py, auto_converter_gui.ps1 파일명은 변경하지 마세요.

6. 오류가 날 때
   - 실행 창의 오류 설명 칸을 먼저 확인합니다.
   - Python 또는 모듈 오류가 나오면 처음실행_필수설치.bat를 다시 실행합니다.
   - PDF 표를 읽지 못한다는 오류가 나오면 PDF가 이미지 스캔본인지 확인합니다.
