나라장터 사전규격 자동 업데이트

1. edit_config.bat을 실행하거나 config.json을 열고 api_key에 공공데이터포털 인증키를 넣습니다.
   - 공공데이터포털에서 "조달청_나라장터 사전규격정보서비스" 활용신청이 필요합니다.
   - 가능하면 "일반 인증키(Decoding)" 값을 넣으세요.

2. run_g2b_prespec_update.bat을 실행하면 다음 작업을 합니다.
   - 마지막 성공 수집일을 기준으로 오늘까지 누락분을 수집합니다.
   - 엑셀에는 "이전 확인 묶음 + 이번 신규 묶음"을 표시합니다.
     예: 19일 실행 시 18~19일을 표시하고 19일 자료를 붉은색 굵은 글씨로 표시합니다.
     예: 20일 미실행 후 21일 실행 시 19~21일을 표시하고 20~21일 자료를 붉은색 굵은 글씨로 표시합니다.
     예: 22일 실행 시 20~22일을 표시하고 22일 자료를 붉은색 굵은 글씨로 표시합니다.
   - 시트명은 2026.05.19. 또는 2026.05.19-05.21 같은 확인 기간 형식으로 생성합니다.
   - 엑셀 파일을 자동으로 엽니다.
   - 사전규격등록번호와 의견등록 마감일시는 API 기본정보에서 채웁니다.
     API 키가 없으면 기존 다운로드 엑셀에 없는 값이라 빈칸으로 남습니다.
   - 사업명을 클릭하면 현재 사용하는 브라우저에서 나라장터 사전규격상세조회 화면이 열립니다.
   - 엑셀 표시 컬럼은 신규, No, 사전규격등록번호, 업무구분, 사업명, 수요기관, 담당자,
     진행일자, 의견등록 마감일시, 진행상태, 예산금액입니다.

3. install_startup_and_shortcut.ps1을 실행하면 다음 작업을 등록합니다.
   - 바탕화면에 "나라장터_사전규격_업데이트" 바로가기를 만듭니다.
   - Windows 작업 스케줄러에 로그온 시 자동 실행 작업을 만듭니다.

4. 사업명 링크를 상세조회까지 연결하는 최초 1회 설정
   - install_g2b_prespec_protocol.ps1을 실행해 g2bprespec:// 링크를 Windows에 등록합니다.
   - install_startup_and_shortcut.ps1을 실행하면 시작프로그램 등록과 함께 위 프로토콜도 자동 등록합니다.
   - 이후 엑셀의 사업명 링크는 로컬 실행 스크립트를 통해 나라장터 상세조회 화면을 엽니다.

5. 사업명 링크가 여는 방식
   - 나라장터는 상세조회 URL만으로 자료가 바로 열리지 않는 SPA 구조입니다.
   - 엑셀의 사업명 링크는 g2bprespec:// 실행 링크를 열고,
     로컬 스크립트가 Chrome에서 나라장터 내부 상세조회 화면을 직접 호출합니다.

6. 일주일 단위로 보고 싶을 때
   - config.json에서 display_mode 값을 week로 바꾸면 이번 주 월요일부터 오늘까지 표시합니다.
   - 기본값은 링크 수를 줄이기 위한 since_last_success입니다.

현재 기본 검색 조건
- 검색유형: 사전규격공개
- 사업명 포함 키워드: 공원, 파크, 조경, 관광, 유원지, 체육, 탐방로, 수목원, 정원, 숲, 휴양림, 치유, 야영장, 레포츠, 산림, 체험, 생태, 경관, 도시재생, 바람길, 둘레길
- 사업명 제외 키워드: 위탁, 건설폐기물, 임차, 감리, 건설사업관리
- 업무구분: 일반용역, 기술용역

참고
- num_of_rows는 999 이하로 두세요. 나라장터 공공데이터 API가 1000으로 요청하면 일부 목록을 10건만 반환하는 경우가 있어 누락이 생깁니다.
