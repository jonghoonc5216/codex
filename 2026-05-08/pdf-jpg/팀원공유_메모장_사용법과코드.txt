포함 파일:
1_처음한번_필요라이브러리_설치.bat
PDF원클릭_JPEG변환.bat

pdf_to_jpeg_코드원본_메모장.txt

처음 사용하는 팀원은:
1. '1_처음한번_필요라이브러리_설치.bat'을 한 번 실행합니다.
2. '바탕화면_바로가기_만들기.bat'을 실행하면 바탕화면에 바로가기가 생깁니다.
3. PDF 파일을 바탕화면의 'PDF원클릭 JPEG 변환' 아이콘 위로 끌어다 놓습니다.

바로 변환하는 방법:
- 'PDF원클릭_JPEG변환.bat'을 더블클릭하면 PDF 선택 창이 뜹니다.
- PDF 파일을 'PDF원클릭_JPEG변환.bat' 위로 끌어다 놓아도 됩니다.
- 여러 PDF를 한 번에 끌어다 놓을 수 있습니다.
- PDF가 들어 있는 폴더를 끌어다 놓으면 그 폴더 안의 PDF 파일을 변환합니다.

결과 저장:
- 원본 PDF가 있는 폴더에 'PDF파일명_jpeg' 폴더가 생성됩니다.
- 여러 페이지 PDF는 'PDF파일명_001.jpg', 'PDF파일명_002.jpg' 형식으로 저장됩니다.

해상도/품질 수정:
- pdf_to_jpeg.py 파일을 메모장으로 열어 아래 값을 바꾸면 됩니다.
- DEFAULT_DPI = 200
- JPEG_QUALITY = 95

필요 조건:
- Windows PC
- Python 설치
- Python 라이브러리 pymupdf, pillow 설치

문제가 생길 때:
- "Python이 설치되어 있지 않습니다"가 나오면 Python을 먼저 설치해야 합니다.
- "필요 라이브러리가 없습니다"가 나오면 '1_처음한번_필요라이브러리_설치.bat'을 실행하세요.
- 회사 보안 정책 때문에 설치가 막히면 전산 담당자에게 Python 패키지 설치 권한을 요청해야 합니다.

핵심 코드는 같은 폴더의 pdf_to_jpeg.py 파일입니다.
메모장으로 코드만 확인하려면 pdf_to_jpeg_코드원본_메모장.txt 파일을 열면 됩니다.
