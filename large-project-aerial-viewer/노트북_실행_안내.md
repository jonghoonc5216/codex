# 조감도작성 노트북 실행 안내

## 가장 쉬운 방법

1. GitHub에서 저장소를 내려받습니다.
2. `large-project-aerial-viewer` 폴더로 이동합니다.
3. `조감도작성_EXE_만들기.bat`를 실행합니다.
4. 빌드가 끝나면 `dist\조감도작성.exe`가 만들어집니다.
5. `조감도작성.exe`를 실행하면 `http://127.0.0.1:8765`가 열립니다.

## 필요한 것

- Windows
- Python
- 인터넷 연결: 처음 한 번 PyInstaller 설치에 필요
- Google Earth Pro: `Google Earth로 열기` 기능 사용 시 필요
- Blender: `Blender로 열기` 기능 사용 시 필요

## Blender 설치 위치

현재 기본 인식 위치:

- `D:\Ai 프로그래밍\Blender\blender.exe`
- `C:\Program Files\Blender Foundation\Blender\blender.exe`

Blender가 다른 위치에 있으면 `server.py`의 `BLENDER_CANDIDATES`에 실행 파일 경로를 추가하면 됩니다.

## GitHub Actions 빌드

GitHub에 올린 뒤 Actions에서 `Build 조감도작성 Windows EXE` 워크플로를 실행하면 `조감도작성-windows-exe` 아티팩트로 exe를 받을 수 있습니다.
