# Codex Utilities

이 저장소는 업무 자동화를 위해 만든 개인용 Codex 도구를 공유하는 공간입니다.

## Sticky Memo Alarm

`sticky-memo-alarm/` 폴더에는 Windows 스티커 메모와 연동되는 보조 프로그램이 들어 있습니다.

주요 기능:

- 스티커 메모에 `알람: 17:30`처럼 적으면 시간이 되었을 때 전체화면 알림 표시
- 검붉은 배경과 흰색 글씨의 전체화면 알림
- 음성 안내 없음
- 스티커 메모의 새 작성/수정 내용을 월별 Word 파일로 자동 기록
- Windows 시작프로그램 자동실행 등록/해제 스크립트 포함

자세한 사용법은 [sticky-memo-alarm/README.txt](sticky-memo-alarm/README.txt)를 참고하세요.
