이 폴더는 PDF 표를 엑셀 산식 결과 파일로 변환하는 자동화 프로그램입니다.

주요 파일
- PDF_엑셀변환_실행.bat: 사용자가 실행하는 파일
- 처음실행_필수설치.bat: 다른 컴퓨터에서 처음 한 번 실행하는 설치 파일
- auto_converter_gui.ps1: Windows Forms 기반 자동변환기 화면
- pdf_formula_converter.py: PDF 표 추출과 엑셀 생성 로직
- template.xlsx: 금액/작업량 시트 서식 템플릿
- requirements.txt: 필요한 Python 모듈 목록

현재 반영된 주요 기준
- 결과 엑셀에는 금액 시트와 작업량 시트만 생성한다.
- 변환기록 시트와 PDF 추출 시트는 생성하지 않는다.
- 작업량 시트 B~N열 열너비는 6.63으로 고정한다.
- 자동변환기 기본 대상면적과 기준면적은 10,000㎡이다.
- 대상면적에 대한 작업량 계수는 (대상면적/기준면적)^(승수)로 계산한다.
- 적용 보정계수는 작업량 계수 x 환산계수1 x 환산계수2 x 적용비율로 계산한다.
- 금액 시트는 template.xlsx의 금액 서식과 열너비를 기준으로 유지한다.

수정 요청 시에는 이 요약과 함께 실제 변경하려는 결과 엑셀 예시 파일을 같이 제공하면 좋습니다.
